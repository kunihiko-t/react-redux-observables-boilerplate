### Provides
- react ^16.2
- react-router ^4.2
- redux ^3.7
- redux-observable ^0.17
- rxjs ^5.5

### Building
- webpack ^3.x

`npm run build`

### Development
- webpack-dev-server ^3.9
- react-hot-loader ^3.1.3

`npm start`

### Tests
- jest ^21.2.1
- enzyme ^3.2

`npm test`  
`npm run test:watch`

### Browser Automation
- nightwatch ^0.9
- selenium ^3.7

`npm run test:ui` (with dev-server already running)  
`npm run test:ui:start` (start dev-server, run tests and exit) 

### Requirements
- node `^7.x`
- npm `^4.x`
